# Imagens das questões

Configuração atual:

- **Questões 1 e 2:** imagens reais (pergunta-1.png, pergunta-2.png) — matriz de formas e grid de círculos.
- **Questões 3 a 25:** placeholders em SVG (pergunta-3.svg … pergunta-25.svg) com o número da questão.

Para usar imagens próprias em uma questão, coloque o arquivo nesta pasta com o nome correto e, se for PNG, altere o campo `imagem` dessa pergunta em `src/data/perguntas.json` para `/questoes/pergunta-N.png`. Formatos aceitos: PNG, JPG, SVG, WebP.
