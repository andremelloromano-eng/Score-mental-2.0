# Imagens do site

Use esta pasta para **imagens gerais** do projeto: logos, ícones, ilustrações, assets de marketing.

## Outras pastas de imagem

- **`public/questoes/`** — Imagens das questões do teste (ex.: pergunta-1.png, pergunta-2.png, …).

## Convenção

- Nomes em minúsculas, sem espaços (ex.: `logo-teste-qi.png`).
- Para ícones/SVG, pode usar também `public/images/icons/` se fizer sentido.
