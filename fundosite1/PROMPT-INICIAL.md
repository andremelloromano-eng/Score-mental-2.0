# Prompt inicial — base do site (fundosite1)

Use este prompt ao criar um **novo site** que tenha a mesma base visual: fundo escuro + partículas flutuantes.

---

## Prompt para colar no Cursor/assistente

```
Cria um site [descreva aqui o tema do site, ex.: landing page, app de quiz, etc.] com:

- Tema escuro (dark mode): fundo em gradiente radial suave (cinza escuro no topo, azul muito escuro #020617 no resto).
- Partículas flutuantes no fundo da página: pequenos pontos que se movem lentamente em várias direções, em tons de cinza ardósia e azul suave, com animação contínua (pode usar Framer Motion). As partículas devem cobrir toda a viewport, atrás do conteúdo, para dar profundidade e um ar profissional.

Mantém o resto do layout simples: conteúdo em cima com z-index acima das partículas, sem mais efeitos. Essa é a base visual que quero reutilizar.
```

---

## Versão curta (só a base)

```
Site em dark mode com fundo em gradiente escuro (#1f2937 no topo, #020617 no resto) e partículas flutuantes em tons de cinza/azul cobrindo toda a página, animadas com Framer Motion. Conteúdo por cima das partículas. Só essa base visual.
```
